# Contributing to Simple Interest Calculator

Thank you for your interest in improving this project! 

All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

## How to Contribute

1. **Fork the Repository:** Create a personal copy of the repository on your GitHub account.
2. **Clone Locally:** Clone the forked repository to your local machine using the Git CLI.
3. **Create a Branch:** Create a descriptively named feature or bug-fix branch.
4. **Make Changes:** Implement your enhancements, bug fixes, or documentation updates.
5. **Commit and Push:** Commit your adjustments with clear messages and push them to your fork.
6. **Open a Pull Request:** Submit a pull request against the main repository for review.

We appreciate your time and effort in helping make this project better!
